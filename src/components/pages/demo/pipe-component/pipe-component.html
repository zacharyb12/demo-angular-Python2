Les pipes

Les pipes sont des fonctionnalités d'Angular qui permettent de transformer les données affichées dans les templates.

Ils sont utilisés pour formater, filtrer ou modifier les données avant de les afficher à l'utilisateur.

Les pipes sont souvent utilisés pour des transformations courantes telles que la mise en majuscule, la mise en minuscule, le formatage des dates, des nombres, des devises, etc.

Angular fournit plusieurs pipes intégrés, mais il est également possible de créer des pipes personnalisés pour répondre à des besoins spécifiques.

Les pipes sont utilisés dans les templates en utilisant le symbole | (pipe) suivi du nom du pipe et de ses éventuels paramètres.

Exemple : pour afficher une date au format court, on peut utiliser le pipe date comme ceci : { { myDate | date : 'short' } }

Création d'un pipe

ng generate pipe pipes/monPipe

Les Pipes Number

Number Pipe

Formate le nombre selon les conventions locales

les paramètres sont :

la premiere valeur est le nombre minimum de chiffre avant la virgule

la deuxieme va définir le nombre minimum de chiffre après la virgule

le troisieme va définir le nombre maximum de chiffre après la virgule

{{myNumber}}
{{myNumber | number }}
{{myNumber | number : '0.0-2'}}
{{myNumber2 | number : '3.0-2'}}
{{myNumber2 | number : '3.2-2'}}
