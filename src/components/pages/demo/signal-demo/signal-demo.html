Signals

Les signaux sont une nouvelle fonctionnalité d'Angular qui permet de gérer la communication entre les composants de manière plus efficace.

Ils permettent de simplifier la gestion des événements et des données partagées entre les composants.
Les signaux sont basés sur le concept de "réactivité", ce qui signifie que les composants peuvent réagir automatiquement aux changements de données sans avoir besoin de gérer manuellement les abonnements ou les événements.

Les signaux sont utilisés pour transmettre des données ou des événements entre les composants, en utilisant un système de publication/abonnement automatique.

Un composant peut émettre un signal lorsqu'un événement se produit, et d'autres composants peuvent s'abonner à ce signal pour recevoir des notifications.
