Details Products

Pour recuperer un produit nous allons pouvoir communiquer avec notre service pour demander un element precis

Nous aurons besoin de l'identifiant du produit

Premièrement nous allons definir un paramètre dans la route : products/:id

Ensuite nous allons utiliser le routeur pour passer l'id en parametre dans l'url

Et dans le composant details nous utiliserons le ActivatedRoute pour recuperer l'id

Ajoutons une route dans le fichier app-routing.module.ts

{path: 'products/:id', component: DetailsProductServComponent}

Ensuite, nous allons utiliser le routeur pour passer l'id en parametre dans l'url

<button (click)="goToDetails(p.id)">Details</button>

Enfin dans le composant details nous utiliserons le ActivatedRoute pour recuperer l'id

private readonly route : ActivatedRoute = inject(ActivatedRoute) const id = +this.route.snapshot.paramMap.get('id')
